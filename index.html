🔢 Sistema Binario

📖 ¿Qué es?

El sistema binario es una forma de representar información usando solo dos números: 0 y 1.

Es el lenguaje básico de las computadoras.

🧠 Ejemplo

Piensa en un interruptor:

💡 Encendido = 1
💡 Apagado = 0

🎯 ¿Para qué sirve?

Sirve para que las computadoras puedan entender y procesar información.

Las computadoras funcionan con electricidad:

  • Hay electricidad → 1
  • No hay electricidad → 0

🧠 Ejemplo

Esa combinación de 1 y 0 es lo que la computadora usa para trabajar.

📍 ¿Dónde se usa?

Se usa en todo lo digital:

  • Computadoras
  • Celulares
  • Videojuegos
  • Internet

🧠 Ejemplo

Todo lo que ves en estos dispositivos está hecho con 0 y 1.

⚙️ ¿Cómo funciona?

Los números binarios funcionan como el sistema normal, pero en base 2.

Cada posición vale el doble que la anterior:

1 → 1
10 → 2
11 → 3
100 → 4

🧠 Ejemplo

1001 en binario:

1×8 + 0×4 + 0×2 + 1×1 = 9

🧠 Ejemplo real

Las letras también se guardan en binario.

La letra A:

01000001

🧠 Ejemplo visual

Eso significa que todo texto, imágenes y videos son combinaciones de 0 y 1.
